CS 124-Lab 1 Walk Through

Task 1: In an empty list, create an item named "Buy new John Grisham"

This is the beginning page of the list application. The user sees a blank list and a button for adding a new item. There are also two buttons at the bottom: one for showing uncompleted items and one for deleting completed list items.

Once the user clicks the "Create new item" text or plus button, a text box appears with a checkbox next to it. The user may begin typing the to-do list item into the text-field.

Once the user fills out the text box, they may hit enter.

After hitting enter, the new item is listed and the option to create a new item button appears again.

Task 2: In a non-empty list, create an item named "Eat Lunch"

This is a page with a preexisting list item.

Once again, the user clicks the plus button or the "Create new item" text and a text field will appear where the user may input the desired list item.

Similar to task 1, the user may hit enter once they have filled out the field with the new list content.

After filling out the list item, the text box is replaced by a new list item and the plus button reappears.

Task 3: Mark the item named "Call Mom" completed

This is a page with multiple preexisting list items.

The user can check the box associated with the list item.

Task 4: Rename the item "Text John" to "Text John about bank statements"

The item "Text John" is listed. The user can hover over the text and click to edit. This is signified with a pencil icon beside the list item.

Once the user clicks a text box appears with the original list item.

The user may directly edit the text field to update the list item.

The text is updated by hitting the enter key. The option to add an item reappears as well along with the pencil icon.

Task 5: Show only uncompleted items.

The user has created a list with some completed and uncompleted list items.

The user clicks the "show uncompleted" items button at the bottom. The completed items are hidden from the user and the "Show uncompleted items" button is replaced with a "Show all items" button which would allow the user to essentially make the completed list items reappear in the same order as before.

Task 6: Delete all completed items.

The user has created a list with some completed and uncompleted list items.

The user clicks the "Delete completed items" button at the bottom. Thee checked list items are then removed permanently.
