Task 1: In an empty list, create an item named "Buy new John Grisham"

This is the beginning page of the list application. The user sees a blank list and a button for adding a new item.

Once the user clicks the "Create new item" button, a text box appears with a checkbox next to it.

Once the user fills out the text box, they may hit enter.

Once the user fills out the text box, a new item is listed and the option to create a new item button appears again.

Task 2

This is a page with a preexisting list item.

Once again, the user clicks the plus button and a text box appears.

Similar to task 1, the user may type enter once they filled out the field with the new list content.

After filling out the list item, the text box is replaced by a new list item and the plus button reappears.

Task 3

This is a page with multiple preexisting list items.

The user can check the box associated with the list item.

Task 4

The item "Text John" is listed. The user can hover over the text and click to edit.

Once the user clicks a text box appears with the original list item.

The user may directly edit the text field to update the list item.

The user edits the list item and the text is updated by hitting the enter key. The option to add an item reappears as well.
